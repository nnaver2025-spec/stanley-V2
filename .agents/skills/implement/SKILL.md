# Implement skill

## Goal
Make the smallest correct change that satisfies the request.

## Procedure
1. Read `AGENT_GUIDE.md`
2. Inspect current code and nearby tests
3. Make a short plan
4. Edit only relevant files
5. Run targeted validation
6. Report:
   - changed files
   - behavior change
   - verification
   - risks
