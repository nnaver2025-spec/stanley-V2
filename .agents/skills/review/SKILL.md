# Review skill

## Checklist
- Is the scope correct?
- Are edge cases covered?
- Are tests sufficient?
- Any side effects?
- Any simpler option?
- Any missing docs or migration notes?

## Output
1. Findings
2. Severity
3. Suggested fixes
4. Validation gaps
